# Ālo Labs Claude Code Plugin Marketplace

Official marketplace for [Ālo Labs](https://alolabs.dev) Claude Code plugins.

## Installation

Add this marketplace to your Claude Code environment:

```
/plugin marketplace add alo-labs/claude-plugins
```

Then install any plugin by name:

```
/plugin install silver-bullet@alo-labs
```
